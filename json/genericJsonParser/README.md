# 通用型 JSON 資料解析器

<br>


---

<br>
<br>

## 說明

<br>

工作上遇到一個工具需求，給入 Json 字串與解析規則，把規則內要的資料濾出并整理成 Map 回傳。

規則限：

1. key 名稱檢查

2. key 的對應值檢查


<br>

要的資料可以指定 Json Object 的 key，或這 Json Array 的 index。

<br>
<br>
<br>
<br>

## 實作

<br>

先定義存放規則的 Java Bean：

<br>

```java
import lombok.Data;

import java.util.HashMap;
import java.util.Map;
@Data
public class FilterRuleSetting {

    // 規則定義，key:String 為規則名稱，val:Map<> 為規則內容
    // 規則內容 Map<目標 key, 預期的 val>
    private Map<String, Map<String, String>> filterRules = new HashMap<>();

    // 需要的目標欄位值 Map<key or index, 回傳時的名稱>
    private Map<Object, String> targetValueMapping = new HashMap<>();

    public void addFilterRules(String ruleName, Map<String, String> rulePair){
        this.filterRules.put(ruleName, rulePair);
    }
}
```

    public Map<String, Object> parse(String jsonStr, FilterRuleSetting setting) {
        JsonElement jsonElement = gson.fromJson(jsonStr, JsonElement.class);
        Map<String, Object> resultMap = new HashMap<>();
        traverseJsonHelper(jsonElement, setting, new LinkedList<>(), resultMap);
        return resultMap;
    }

    private void traverseJsonHelper(JsonElement jsonElement, FilterRuleSetting setting, LinkedList<JsonElement> stack, Map<String, Object> resultMap) {
        while (true) {
            if (jsonElement.isJsonObject()) {
                JsonObject jsonObject = (JsonObject) jsonElement;

                objectFilter(jsonObject, setting, resultMap);

                for (String key : jsonObject.keySet()) {
                    stack.addLast(jsonObject.get(key));
                }
            } else if (jsonElement.isJsonArray()) {
                JsonArray jsonArray = (JsonArray) jsonElement;
                for (JsonElement element : jsonArray) {
                    stack.addLast(element);
                }
            }

            if (stack.isEmpty()) {
                break;
            } else {
                jsonElement = stack.removeFirst();
            }
        }
    }

    private void objectFilter(JsonObject jsonObject, FilterRuleSetting setting, Map<String, Object> resultMap) {
        Map<String, Map<String, String>> nameRolesPair = setting.getFilterRules();
        // 規則名稱集合
        Set<String> ruleNames = nameRolesPair.keySet();

        for (String ruleName : ruleNames) {
            Map<String, String> rules = nameRolesPair.get(ruleName);

            Set<String> ruleKeys = rules.keySet();

            boolean checkFlag = true;
            boolean isSub = false;
            JsonElement subElement = null;

            for (String ruleKey : ruleKeys) {
                JsonElement jsonElement = jsonObject.get(ruleKey);
                // key 檢查
                if (jsonElement == null) {
                    checkFlag = false;
                    break;
                } else {
                    // value 檢查
                    String checkVal = rules.get(ruleKey);
                    if (checkVal == null || checkVal.equals("")) {
                        subElement = jsonElement;
                        isSub = true;
                        break;
                    }
                    String objVal = jsonElement.getAsString();
                    if (!checkVal.equals(objVal)) {
                        checkFlag = false;
                        break;
                    }
                }
            }
            if (checkFlag) {
                if (isSub) {
                    processJsonElement(subElement, ruleName, setting.getTargetValueMapping(), resultMap);
                } else {
                    processJsonElement(jsonObject, ruleName, setting.getTargetValueMapping(), resultMap);
                }
            }
        }
    }
